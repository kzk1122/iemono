# Iemono - おうちにあるものまとめて管理ツール

おうちにあるものをかんたんにまとめて管理するWebアプリケーションです。

## 技術スタック

- Next.js 16 (App Router)
- React 19
- TypeScript
- Tailwind CSS v4
- localStorage (データ永続化)

## 機能一覧

### ダッシュボード (`/`)
- 総アイテム数、期限注意、期限切れの統計表示
- 賞味期限タイムライン（食品・薬のみ、期限が近い順に表示）
- カテゴリ別アイテム数
- 保管場所別アイテム数

### アイテム一覧 (`/items`)
- 登録済みアイテムの一覧表示
- アイテム名による検索
- カテゴリによるフィルタリング（食品、日用品、家電、衣類、薬・医療品、その他）
- 保管場所によるフィルタリング（冷蔵庫、冷凍庫、パントリー、キッチン、洗面所、クローゼット、リビング、その他）
- 各アイテムにカテゴリ・保管場所・数量・単位・賞味期限・メモを表示
- 賞味期限の残り日数をバッジで表示（食品・薬のみ）
- 一覧から直接編集・削除が可能

### アイテム追加 (`/items/new`)
- アイテム名、カテゴリ、保管場所、数量、単位、メモを入力
- カテゴリ選択: 食品🍱、日用品🧴、家電🔌、衣類👕、薬・医療品💊、その他📦
- 数量は +/- ボタンで操作、単位は自由入力
- 賞味期限・通知設定は食品と薬・医療品のみ表示
- 賞味期限はカレンダーピッカーから選択（今日ボタン・クリア機能付き）
- アイテムごとに通知タイミングを設定可能（1日前〜30日前）

### アイテム編集 (`/items/edit?id=xxx`)
- 既存アイテムの情報を編集

### 設定 (`/settings`)
- テーマ切り替え（ライト / ダーク / システム）
- 賞味期限通知のON/OFF切り替え
- デフォルト通知タイミングの選択（1日前〜30日前）
- 期限切れアイテムの表示/非表示
- 全データの削除

### 通知バナー
- 期限切れ・期限間近のアイテムがある場合、画面上部にバナーを表示

### ボトムナビゲーション
- ホーム、一覧、追加、設定の4画面をワンタップで切り替え

### PC表示対応
- スマホアプリ風のコンテナレイアウト（中央に最大幅制限）
- サイドにグラデーション背景と装飾イラストを表示
- ライト/ダークモードに応じたサイド画像の自動切り替え

## セットアップ

```bash
npm install
npm run dev
```

[http://localhost:3000](http://localhost:3000) をブラウザで開いてください。
