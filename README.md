# Iemono - 食品管理アプリ

家庭の食品在庫をかんたんに管理するWebアプリケーションです。

## 技術スタック

- Next.js 16 (App Router)
- React 19
- TypeScript
- Tailwind CSS v4
- localStorage (データ永続化)

## 機能一覧

### ダッシュボード (`/`)
- 登録アイテムの合計数、期限切れ数、期限間近の数を表示
- 保管場所ごとのアイテム数を一覧表示

### 食品一覧 (`/items`)
- 登録済み食品の一覧表示
- 食品名による検索
- 保管場所によるフィルタリング（冷蔵庫、冷凍庫、パントリー、キッチン、その他）
- 各アイテムに保管場所・数量・賞味期限・メモを表示
- 賞味期限の残り日数をバッジで表示（期限切れ: 赤、期限間近: 黄、OK: 緑）
- 一覧から直接編集・削除が可能

### 食品追加 (`/items/new`)
- 食品名、数量、保管場所、賞味期限、メモを入力
- 数量は +/- ボタンで操作
- 保管場所はチップ選択式
- 賞味期限はカレンダーピッカーから選択（今日ボタン・クリア機能付き）

### 食品編集 (`/items/[id]/edit`)
- 既存アイテムの情報を編集

### 設定 (`/settings`)
- 賞味期限通知のON/OFF切り替え
- 通知タイミングの選択（1日前〜30日前）
- 全データの削除

### 通知バナー
- 期限切れ・期限間近のアイテムがある場合、画面上部にバナーを表示

### ボトムナビゲーション
- ホーム、一覧、追加、設定の4画面をワンタップで切り替え

## セットアップ

```bash
npm install
npm run dev
```

[http://localhost:3000](http://localhost:3000) をブラウザで開いてください。
